# EPG-LearnLink – Plateforme pédagogique

## Description

Ce projet a été réalisé dans le cadre d’un stage à l’École Polytechnique des Génies (EPG).

L’objectif du projet est de concevoir et développer une plateforme web pédagogique permettant de centraliser les supports de cours (cours, TD, TP, corrigés) et de faciliter leur gestion et leur consultation par les étudiants et les enseignants.

La plateforme permet également à l’administration de gérer les utilisateurs, les formations et les modules via un tableau de bord.

---

## Fonctionnalités principales

- Authentification des utilisateurs (administrateur, enseignant, étudiant)
- Gestion des formations et modules
- Dépôt et gestion des supports pédagogiques
- Consultation et téléchargement des ressources
- Évaluation et commentaires des supports
- Tableau de bord administratif
- Statistiques de consultation des ressources

---

## Architecture du projet

Le système repose sur une architecture **3-tiers** :

- Couche présentation : interface web
- Couche logique métier : Laravel
- Couche données : MySQL

L’application suit également le modèle **MVC (Model View Controller)**.
